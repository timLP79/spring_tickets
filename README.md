# Spring Tickets

A ticketing system built with Spring Boot and Thymeleaf.  
This project is currently in early development.

## Features (Planned)
- Create and assign tickets
- Track status and due dates
- Prioritize tasks
- Filter and search

## Tech Stack
- Java 21
- Spring Boot
- Thymeleaf
- Gradle
